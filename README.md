# Abdul Basit — Personal Profile

A non-conventional personal profile site for **Abdul Basit**, Power Platform Architect · Dynamics 365 CE · Copilot Studio builder.

## Pages
- **`index.html`** — Profile / CV (bento-grid layout)
- **`articles.html`** — Articles & writing (Medium: [@powerdynamite](https://medium.com/@powerdynamite))
- **`license-planner.html`** — Interactive Power Platform & D365 licensing advisor (role-based cost estimator)
- **`contact.html`** — Consultancy booking, rates, testimonials & FAQ

## Tech
Plain static HTML/CSS/JS — no build step.
- `css/` — modular stylesheets (tokens → base → layout → components → pages)
- `js/` — page interactions

## Run locally
```bash
npx serve .
```
Then open http://localhost:3000

## Deploy
Hosted on **GitHub Pages** — pushing to `main` publishes automatically.
